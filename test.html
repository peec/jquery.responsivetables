Responsive tables

This is a jQuery plugin. The plugin makes it possible to add a selector to tables that makes the tables responsive. Add forexample $('table').responsivetable(); to your javascript

Download:

  • Download v0.0.1

Requirements:

  • enquire.js

Example usage:

$(function () {
	$('table').responsivetables();
});

Styles:

The plugin automaticaly adds responsive-table class to matched tables. .mobile- prefixes are only applied in on small screens.

.responsive-table .mobile-tr {
	display: block; /* This is probably always great to have.*/
}
.responsive-table .mobile-th {
}
.responsive-table .mobile-td {
}
